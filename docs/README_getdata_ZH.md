## `get_csv_data.py`程序说明

### 功能

该程序用于读取并处理 CSV 格式的数据文件，提取儒略日（Julian Dates）、光子流量（Photon Fluxes）及光子流量误差（Photon Fluxes Error）数据，同时处理异常数据并提取源名称。

### 输入文件格式要求

1. **文件类型**：标准 CSV 格式文件，后缀为`.csv`

2. **表头要求**：文件第一行必须为表头行，程序会将第一行作为列名读取

3. **数据列要求**：

    - 第 2 列（索引为 1）必须为儒略日数据

    - 第 5 列（索引为 4）必须为光子流量数据，若该列数据为字符串类型，程序会过滤掉包含`<`符号的行

    - 第 6 列（索引为 5）必须为光子流量误差数据

4. **空值表示**：文件中空值需用`-`表示，程序会将其识别为缺失值

5. **文件名要求**：文件名需采用`[源名称1]_[源名称2]_[源名称3]_xxx.csv`的格式，程序会通过下划线分割文件名的前三个部分作为源名称

> [!Tip]
> 可以在程序中调节索引来使程序匹配您的程序！

### 返回值

程序返回四个值：

1. `source_name`：提取的源名称

2. `julian_dates`：处理后的儒略日数据数组

3. `photon_fluxes`：处理后的光子流量数据数组

4. `photon_fluxes_err`：处理后的光子流量误差数据数组

## `get_txt_data.py`程序说明

### 功能

该程序用于读取并处理 TXT 格式的数据文件，提取儒略日（Julian Dates）、光子流量（Photon Fluxes）及光子流量误差（Photon Fluxes Error）数据，同时处理注释行、空行和异常数据并提取源名称。

### 输入文件格式要求

1. **文件类型**：纯文本格式文件，后缀为`.txt`

2. **行处理规则**：

    - 第一行会被直接跳过（无论是否为注释行）

    - 以`#`开头的行被视为注释行，会被跳过

    - 空行会被跳过

3. **数据列要求**：

    - 每行至少包含三列数据，列之间用空格或制表符分隔

    - 第 1 列必须为儒略日数据（需为可转换为浮点数的数值）

    - 第 2 列必须为光子流量数据（需为可转换为浮点数的数值）

    - 第 3 列必须为光子流量误差数据（需为可转换为浮点数的数值）

4. **文件名要求**：文件名需采用`[源名称1]_[源名称2]_[源名称3]_xxx.txt`的格式，程序会通过下划线分割文件名的前三个部分作为源名称

### 返回值

程序返回四个值：

1. `source_name`：提取的源名称

2. `julian_dates`：处理后的儒略日数据数组

3. `photon_fluxes`：处理后的光子流量数据数组

4. `photon_fluxes_err`：处理后的光子流量误差数据数组

## 使用注意事项

1. 两个程序都会自动过滤掉包含`nan`的异常数据

2. 程序会将处理过的文件名记录到传入的`state`字典的`processed_files`列表中

3. 若 CSV 文件为空，程序会返回空的 DataFrame 并提示文件为空已跳过

4. 文件名的格式必须符合要求，否则可能导致源名称提取错误

5. 确保输入文件的列顺序和数据类型符合要求，否则可能导致程序运行错误
